CAPSTONE PROJECT

Betterflye

GitHub Repository


Tech Stack:

PHP, HTML, JavaScript, SQL, Ajax

PHPStorm, MySQL


In a team with three other Ball State students, I created a system for the start-up social media company Betterflye to control the content posted by their users. This includes a filter that automatically detects profanity and prevents it from being posted publicly on the platform. This project also included creating tools for the website admin, such as managing posts flagged by users and the ability to delete users that have uploaded profane content.

CS222 PROJECT

Perfect Plan

Tech Stack:

Java, XML, JavaFX

Intellij IDEA


In a team with three other students, I created an application for Ball State students to make the most of their meal plans. The user can track how many meal swipes they have left in their plan and view a list of meals that cost exactly $8.85. The user can add their own custom meals. Both custom and default meals are separated by dining location, and can be further filtered based on dietary restrictions such as dairy, gluten, or meat-free.
